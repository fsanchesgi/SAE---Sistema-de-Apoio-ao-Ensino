Política de Privacidade

O SAE – Sistema de Apoio Educacional respeita a privacidade e a proteção dos dados pessoais de todos os usuários.

Coleta de dados

Coletamos apenas os dados necessários para o funcionamento da plataforma, incluindo informações cadastrais, acadêmicas e institucionais.

Uso das informações

Os dados são utilizados exclusivamente para fins educacionais, administrativos, operacionais e de melhoria contínua dos serviços.

Compartilhamento

O SAE não vende ou comercializa dados pessoais. O compartilhamento ocorre apenas quando necessário para cumprimento de obrigações legais ou técnicas.

Segurança

Adotamos medidas técnicas e organizacionais para proteger os dados contra acessos não autorizados, vazamentos ou perdas.

Alterações

Esta política pode ser atualizada a qualquer momento. Recomendamos a revisão periódica deste documento.
